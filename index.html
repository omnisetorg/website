Quick Install

For a base installation with essential tools, run:

wget -qO- https://omniset.org/install | bash

Or choose a profile below for a customized setup.

Installation Options

1

Choose Your Setup

Run the base installation for essentials, pick a pre-built profile, or customize your own setup with specific tools.

2

Install With One Command

Use the standard command for base install, or add ?profile=dev for a specific profile. Example: ?profile=gamer

3

Customize Further

After installation, you can add more tools or modify your setup anytime using the included management tools.

Pre-built Profiles

Choose from our curated profiles or use them as a starting point for your custom setup. Add "?profile=NAME" to the installation command to use a specific profile.
