X is a script that automatically sets up a complete web development environment on Debian-based Linux systems with a single command.
X works on any Debian-based Linux distribution including Ubuntu, Debian, Raspberry Pi OS, Linux Mint, Elementary OS, Pop!_OS, and Zorin OS.
We want X to be as stable and intuitive as possible. Most people don't care about Arch, kernels, and technical details - they just want a system that works reliably.
X installs essential development tools (Git, Zsh, curl, wget), Node.js, VS Code, Chrome, and optional tools like VLC.
All installation scripts are open source and available for review. We only install packages from official repositories or verified sources.
wget -qO- https://x.org/install | bash
wget -qO- https://x.org/install?tools=base,node,vscode,chrome | bash
